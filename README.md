🚀 **First Internship Project Completed – Digital Voting System in Python** 🗳️

I'm excited to share my first mini project completed during my **M Tech Summer Internship**.

As part of **Task 1**, I developed a **Digital Voting System** using Python. This console-based application simulates a simple electronic voting process, allowing users to cast votes securely, preventing duplicate voting through unique voter IDs, displaying election results, and automatically declaring the winner.

### 🔹 Key Features

* Secure voting using unique Voter IDs
* Duplicate vote prevention
* Real-time vote counting
* Election results display
* Automatic winner declaration
* User-friendly menu-driven interface
